# 🗺️ BD Heritage Map

## 📌 Overview

An interactive map-based web application that allows users to explore GI-certified products of Bangladesh by district.

## 🎯 Goal

To digitally preserve and promote Bangladesh’s geographical heritage and unique products.

## 🛠️ Technologies Used

* JavaScript
* HTML
* CSS

## ⚙️ Features

* Interactive district-based map
* GI product discovery
* Informative content for each location

## 🚀 How to Run

Open `index.html` in your browser.
